# Typography And Composition

Most "cheap" ad output is not an animation problem. It is a still-frame problem: weak type hierarchy, cramped spacing, lazy fonts, unclear visual weight, or too many ideas fighting in the frame.

Use this reference before adding motion to premium, Apple-like, cinematic, luxury, technical, or "less template" videos.

## Hero Frame Test

For every scene, pick one hero frame timestamp. At that timestamp:

- The main message must be readable without replaying.
- One element must clearly dominate.
- Secondary text must support, not compete.
- Product/screenshot/proof must have enough size and contrast.
- Safe margins must hold for the target platform.
- Decorative layers must not carry more visual weight than the message.

If the hero frame does not work as a static poster, fix layout before motion.

## Type Scale

Use deliberate contrast:

- Display title: large enough to carry the frame.
- Subtitle: visibly secondary, usually 35-55% of title size.
- Proof/stat: either dominant as a number or subordinate as support, not halfway.
- Legal/helper text: readable but quiet.

Avoid too many sizes. Three levels are usually enough for a short ad.

## Letter Spacing And Line Height

Default letter spacing should be `0`. Add letter spacing only for a specific brand reason or all-caps micro-labels.

Use line height to create polish:

- Large display text: tight but not colliding.
- Multi-line title: enough spacing to read as a designed block.
- Captions/subtitles: comfortable, not compressed.

If a word is too long for the container, change layout, width, or copy before shrinking everything.

## Font Choice

Avoid lazy defaults when the user asks for high-end visuals.

Acceptable choices depend on brand, but document the reason:

- Premium/product launch: neutral grotesk, strong display sans, restrained system fallback.
- Technical/SaaS: clean sans plus optional monospace accent.
- Luxury: high-contrast display or elegant serif only when it fits the product.
- Social/hype: heavier display face, but keep hierarchy disciplined.

Do not mix more than two font families unless the brand requires it.

## Composition Rules

- Use negative space as a design element.
- Keep one dominant visual mass per beat.
- Align elements intentionally: center, grid, or edge system.
- Do not decorate empty space just because it is empty.
- Do not put large text, screenshot, stats, and CTA at equal weight in the same hero frame.
- For product screenshots, add enough contrast, framing, or scale so the viewer knows where to look.

## Platform Safe Areas

### 16:9

- Horizontal breathing room can create premium feel.
- Product frames can sit center or slightly below center.
- Avoid placing CTA too close to bottom edge.

### 9:16

- Keep central visual mass.
- Account for app UI overlays.
- Reduce words per beat.
- Avoid wide screenshot layouts unless cropped deliberately.

## Cheapness Diagnostic

When the user says "cheap", inspect in this order:

1. Too many effects on type.
2. Weak type scale.
3. Bad font pairing.
4. Cramped line breaks.
5. Low contrast or muddy gradient.
6. Repeated template motion.
7. Decorative glow/particles overpowering message.
8. No product-specific visual moment.

Fix the first real cause. Do not add more effects to hide weak composition.
